# Business Sales Data Analysis

## Business Context
A retail company wants to understand sales trends, top products, and customer behavior to improve profitability.

## Objective
Analyze sales data to identify key trends, profitable products, and customer segments.

## Dataset
- Source: Kaggle “Superstore Sales”
- Columns: Order Date, Product, Region, Sales, Profit, Customer Segment

## Key Questions
1. Which products sell best?
2. Which regions generate the most revenue?
3. Which months/quarters are peak sales?
4. Which customer segments are most valuable?
5. Where are profit opportunities?
